Lenguajes de programacion

Los lenguajes de programacion son un conjunto de grafemas que son usados para dar ordenes a una computadora, existen varios tipos de lenguajes con sus propias reglas y sintaxis que se deben seguir para lograr con exito la compilacion/interpretacion por la maquina

Paradigmas

Existen diferentes tipos de paradigmas en programacion, un paradigma es una forma de pensar al momento de programar, pueden dividirse en 2 tipos:

Programacion imperativa

  • Programacion estructurada

    Es la mas basica, esta enfocada en el control del flujo de ejecucion como las declaraciones if que saltan lineas si una condicion no se cumple o los bucles que regresan y repiten lineas de codigo

  • Programacion procedimental

    Aqui, el programa se divide en funciones, en donde cada una realiza un trabajo diferente, esto permite la modularidad que a su vez evita la repeticion de codigo

  • Programacion Orientada a Objetos

    En este paradigma, la abstraccion toma mas peso que en el procedimental, todo es tratado como un objeto, teniendo cada uno sus propios atributos y funciones

Programacion declarativa

  • Programacion funcional

    Se centra en el uso de funciones, la diferencia con la procedimental radica en que esta generalmente no devuelve datos al momento de llamar a la funcion

Java

Java es un lenguaje de programacion desarrollado por James Goslin, usa los paradigmas de OOP, procedimental, estructurada y funcional su entorno de ejecucion es el jre (java runtime enviroment) en donde posee un recolector de basura integrado, (elimina objetos si estos ya no estan referenciados en ningun lugar) el jdk (java devlopment kit) es una utilidad que permite crear aplicaciones java, viene con el compilador y el jre el codigo compilado por java es por defecto compilado en bytecode, un codigo intermedio que es interpretado por el jvm

Protocolos de red

los protocolos, son reglas y pasos que se deben seguir para completar con exito algun objetivo, en el caso de las redes implica lograr una conexion exitosa entre 2 dispositivos finales, existen varios protocolos separados principalmente en capas, hablaremos de la capa aplicacion:

    Capa aplicacion

    Esta capa es el intermediario entre el usuario y la red, aqui exiten una gran variedad de ellos, entre las principales tenemos:

    • SMTP: puerto 25, (Simple Mail Tranasfer Protocol) gestor de correos
    • HTTP: puerto 80, (Hyper Text Transfer Protocol) usado para transportar paginas web
    • HTTPS: puerto 443, (Hyper Text Transfer Protocol Secure) usado para transportar paginas web, implementa TLS o SSL
    • FTP: puerto 20 - 21, (File Transfer Protocol) usado para transferir archivos

Base binaria

Las computadoras solo entienden 2 estados, encendido, apagado, true, false, 1,0, todo numero puede ser representado en base 2
